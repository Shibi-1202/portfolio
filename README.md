# Portfolio Website

A minimal, single-page portfolio website with smooth scrolling.

## File Structure

```
portfolio/
├── src/
│   ├── index.html         # Main HTML file
│   ├── styles.css         # All CSS styles
│   └── script.js          # JavaScript functionality
├── assets/
│   ├── favicon.ico        # Custom favicon
│   ├── images/            # Project images
│   │   ├── project1.jpg
│   │   ├── project2.jpg
│   │   └── project3.jpg
│   ├── certificates/      # Achievement certificates
│   │   ├── cert1.jpg
│   │   ├── cert2.jpg
│   │   └── cert3.jpg
│   └── logo/              # Logo files
│       └── logo.png
└── README.md
```

## Sections

1. **Hero** - Introduction with name and tagline
2. **Skills & Competences** - Display your technical skills
3. **Projects** - Showcase your work with images and links
4. **Achievements** - Display certificates and awards
5. **Contact** - Contact form and social links

## Setup

1. Replace placeholder content in `src/index.html` with your information
2. Add your images to the `assets/` folders:
   - Project images → `assets/images/`
   - Certificates → `assets/certificates/`
   - Logo → `assets/logo/`
3. Replace `assets/favicon.ico` with your custom favicon
4. Open `src/index.html` in a browser

## Customization

- **Colors**: Edit CSS variables in `src/styles.css` (`:root` section)
- **Content**: Update text in `src/index.html`
- **Styling**: Modify `src/styles.css`
- **Functionality**: Add features in `src/script.js`

## Features

- Smooth scrolling navigation
- Responsive design (mobile-friendly)
- Fade-in animations on scroll
- Contact form with validation
- Clean, minimal design

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
